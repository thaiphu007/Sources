ReadEAN13 

Overview

When set to TRUE the toolkit will search for EAN-13 type barcodes.  

Default value =  TRUE 

void STSetParameter(hBarcode, ST_READ_EAN13, uint16 *value) ;

void STGetParameter(hBarcode, ST_READ_EAN13, uint16 *value) ;

e.g

// Turn off EAN-13 recognition.
#include "barcode.h"
uint16 value = 0 ;
STSetParameter(hBarcode, ST_READ_EAN13, &value) ;
