ReadDataMatrix 

Overview

When set to TRUE the toolkit will search for data matrix barcodes.  

Default value =  FALSE 

void STSetParameter(hBarcode, ST_READ_DATAMATRIX, uint16 *value) ;

void STGetParameter(hBarcode, ST_READ_DATAMATRIX, uint16 *value) ;

e.g

// Turn on data matrix recognition.
#include "barcode.h"
uint16 value = 0 ;
STSetParameter(hBarcode, ST_READ_DATAMATRIX, &value) ;
