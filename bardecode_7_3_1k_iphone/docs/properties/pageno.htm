PageNo 

Overview

PageNo is a 1 based index that specifies the page to be scanned in an image. A value of zero indicates that every page will be scanned  

Default value =    0

void STSetParameter(hBarcode, ST_PAGE_NO, uint16 *value) ;

void STGetParameter(hBarcode, ST_PAGE_NO, uint16 *value) ;

e.g

// Only check for barcodes on page 3
#include "barcode.h"
uint16 value = 3 ;
STSetParameter(hBarcode, ST_PAGE_NO, &value) ;
