ReadCodabar 

Overview

When set to TRUE the toolkit will search for type 39 barcodes.  

Default value =  TRUE 

void STSetParameter(hBarcode, ST_READ_CODABAR, uint16 *value) ;

void STGetParameter(hBarcode, ST_READ_CODABAR, uint16 *value) ;

e.g

// Turn off Codabar recognition
#include "barcode.h"
uint16 value = 0 ;
STSetParameter(hBarcode, ST_READ_CODABAR, &value) ;
