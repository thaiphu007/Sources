ReadMicroPDF417

Overview

When set to TRUE the toolkit will search for micro-PDF-417 barcodes.

Default value =  FALSE

void STSetParameter(hBarcode, ST_READ_MICRO_PDF417, uint16 *value) ;

void STGetParameter(hBarcode,  ST_READ_MICRO_PDF417, uint16 *value) ;

e.g

#include "barcode.h"
uint16 value = 1 ;
STSetParameter(hBarcode, ST_READ_MICRO_PDF417, &value) ;
