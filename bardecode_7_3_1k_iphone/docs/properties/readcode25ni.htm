ReadCode25ni 

Overview

When set to TRUE the toolkit will search for type 2 of 5 non-interleaved barcodes.  

Default value =   FALSE

void STSetParameter(hBarcode, ST_READ_CODE25_NI, uint16 *value) ;

void STGetParameter(hBarcode, ST_READ_CODE25_NI, uint16 *value) ;

e.g

// Turn on Code 25 non-interleaved recognition.
#include "barcode.h"
uint16 value = 1 ;
STSetParameter(hBarcode, ST_READ_CODE25_NI, &value) ;
